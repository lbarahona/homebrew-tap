# Homebrew Tap

Homebrew formulae for tools by [@lbarahona](https://github.com/lbarahona).

## Installation

```bash
brew tap lbarahona/tap
```

## Available Formulae

### Argus

AI-powered observability CLI for SREs. Query logs, traces, and metrics from Signoz with natural language analysis.

```bash
brew install lbarahona/tap/argus
```
